Los tres lenguages del Desarrollo Web

HTML

  • ⚫Hyper-Text Markup Language por sus siglas en inglés
  • ⚫Es un lenguaje de MARCADO DE TEXTO
  • ⚫Sirve como estructura de la pagina web
  • ⚫Usa etiquetas para marcar texto y así ordenarlo

Javascript

  • ⚫Abreviado comunmente como JS
  • ⚫Es un lenguaje de PROGRAMACIÓN INTERPRETADO
  • ⚫Sirve para enviar y recibir información del servidor de una página web
  • ⚫Entiende de forma nativa a los navegadores

CSS

  • ⚫Cascade Style Sheet por sus siglas en inglés
  • ⚫Es un lenguaje de DISEÑO GRÁFICO
  • ⚫Sirve para definir la presentación de la pagina web
  • ⚫Se vale de un sistema de cascada para definir los estilos y el layout de la página
